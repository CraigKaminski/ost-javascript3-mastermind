Mastermind

Welcome to my JavaScript version of Mastermind. For those not familiar with Mastermind, the goal is to break a code generated by the web browser. The code is comprised of four colors. The possible colors are yellow, brown, red, purple, blue, and green. A color can be used more than once in the code. Each time you make an attempt, a key to the right of your attempt will indicate how close you are to solving the code. The number of black circles indicate how many colors you had in the correct positions. The number of white circles indicate how many colors were correct, but were not in the correct position.
